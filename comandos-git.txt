# Comandos para subir el proyecto a GitHub
# ==========================================

# Paso 1: Verificar que estamos en el directorio correcto
# cd "d:\Proyectos Mios\maxsalud"

# Paso 2: Agregar todos los archivos al staging area
git add .

# Paso 3: Ver qué archivos se van a subir (opcional)
git status

# Paso 4: Crear el primer commit
git commit -m "Initial commit: MaxSalud API - Sistema de reportes de pacientes"

# Paso 5: Cambiar el nombre de la rama a main (si es necesario)
git branch -M main

# Paso 6: Conectar con el repositorio remoto de GitHub
# IMPORTANTE: Reemplaza 'tu-usuario' con tu usuario de GitHub
git remote add origin https://github.com/tu-usuario/maxsalud-api.git

# Paso 7: Subir el código a GitHub
git push -u origin main

# ==========================================
# Notas importantes:
# - El archivo .gitignore ya está configurado para NO subir:
#   * El directorio wallet/
#   * El archivo application.properties (si está descomentado en .gitignore)
#   * Archivos compilados (target/)
# - Verifica que NO se suban contraseñas ni información sensible
# ==========================================
